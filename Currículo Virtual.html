Projetos e Experiências

Projeto de infraestrutura IoT e arquitetura de redes para empresas

Projeto realizado no 2º ano do Ensino Médio Técnico em IoT, cujo objetivo foi elaborar uma apresentação profissional detalhada para a montagem de uma infraestrutura de redes e IoT. A proposta incluía os equipamentos necessários, diferentes links de internet (com base no custo-benefício), especificações técnicas e valores, com o objetivo de automatizar a quadra e a praça de alimentação do Senac Nações Unidas. Todo o projeto foi apresentado à professora responsável, envolvendo planejamento detalhado e análise de alternativas para o cliente, oferecendo soluções práticas e personalizadas.
Neste trabalho foi necessário o aprimoramento do trabalho em grupo, segue o anexo do trabalho de quando ainda estava em processo de realização (Baixar PDF).

Implementação de Sensor de Proximidade em Bonés para Deficientes Visuais

No 3º ano do Ensino Médio Técnico, participei da implementação de sensores de proximidade em bonés para auxiliar deficientes visuais. Este projeto integrou conceitos de Internet das Coisas (IoT) com a inclusão social. Utilizamos LEDs, um sensor de proximidade, uma placa Arduino, fios, conectores e uma protoboard para realizar a montagem. Além disso, desenvolvemos o código necessário utilizando o aplicativo Arduino IDE, garantindo o funcionamento adequado do dispositivo. Este projeto reforçou a importância de inovação tecnológica com foco em acessibilidade.



Criação de carrinho seguidor de linhas

No 3º bimestre do 3º ano do Ensino Médio Técnico, participei de um projeto que consistia na criação de um carrinho seguidor de linhas. Utilizamos um kit de robótica Arduino, incluindo uma placa Arduino, motores, rodas, chassi e sensores de linha. Após a montagem do hardware, desenvolvemos o código necessário no software Arduino IDE para que o carrinho seguisse trajetos pré-determinados. Este projeto exigiu habilidades de lógica de programação, trabalho em equipe e integração de diferentes áreas do conhecimento, como eletrônica e programação.
