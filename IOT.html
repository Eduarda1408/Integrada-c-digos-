Meus Projetos de IoT

Aqui você pode ver alguns dos projetos que realizei durante o curso técnico de IoT. Estes projetos foram desafiadores e me ajudaram a aprender novas habilidades.

Projeto 1: Boné com sensor

Para finalizar o segundo bimestre nos reunimos em um grupo de quatro pessoas com o objetivo de juntar todos os códigos de programação que estudamos ao decorrer desse processo, e para que isso fosse possível o professor preparou uma atividade que se consiste na construção de um boné com sensores sonoros e visuais, e pensando nisso separamos, montamos, testamos os circuitos e códigos e integramos com as outras áreas do conhecimento (Linguagens, Matemática, Ciências Humanas e Ciências da Natureza) para a realização dessa tarefa e documentamos tudo em um arquivo.

Projeto 2: Carrinho seguidor de linhas com Arduino

O principal objetivo do projeto é o desenvolvimento de um carrinho seguidor de linha que utilizando a plataforma de Arduino, possa mandar comandos para movimentação do mini automóvel. A proposta é a construção física do carrinho e a programação para controlar os sensores, dando ao carrinho movimento e comandos de resposta.

Projeto 3: Lixeira com abertura automática

Este projeto foi o penúltimo que realizamos, nele utilizamos novamente o kit disponibilizado pela escola, assim como em todos os projetos que foram realizados. Seu processo de criação se deu a partir da junção dos fios jumpers no sensor ultrassônico, ligamos os fios no Arduino, utilizamos o servo motor, etc. O resultado ficou incrível e fácil de se utilizar!

Projeto 4: Dispenser para ração pet

Este projeto foi o último que realizei com meu grupo da escola, um dispenser pensando na facilidade de servir comida para os nossos pets, criamos ele com um código que lançamos no Arduino IDE e a montagem física, além de montar a estrutura com papelão e decorar.

Armários do Senac: Aulas práticas

O senac possui um armário em cada sala do técnico, neles ficam todos os materiais que os alunos podem utiizar, além de se encontrar os kits arduinos nele. Os kits são muito preservados e fácil de usar, isso pois o colégio tem todo o cuidado de manter uma folha sulfitee com a organização dos tens com todos os kits, fazendo com que ao guardar os alunos não mudem a ordem.Além dos kits este armário é utilizado para guardar todos os trabalhos dos estudantes e para guardar o restante dos materiais que serão utiizados pelos futuros alunos.
